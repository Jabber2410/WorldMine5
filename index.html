Was ist WorldMine 5?

World Mine 5 ist ein Minecraft Projekt, ähnlich zu Craftattack. Diese Season gibt es Mods auf dem Server. Wir spielen auf einem Forge Server in der 1.20.1 Version. Jeder darf mitspielen und man braucht nur einen Java Account mit einem geeignetem Modloader. Um mitzuspielen, trete unserem Discord Server bei und lies dort die Informationen.
Viel Spaß beim Spielen.
